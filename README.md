# メール番人

Gmailの誤送信を防ぐ番人。送信前に宛先・添付・誤字脱字・日付をチェックするChrome拡張機能です。

## 機能

### 1. 送信元チェック
- 送信元アカウントを確認表示

### 2. 送信先チェック
- To/Cc/Bccの宛先を一覧表示
- 重複アドレスの検出
- 大量送信時の警告

### 3. 添付ファイルチェック
- 「添付」「ファイル」などのキーワードがあるのに添付がない場合に警告
- 添付ファイル一覧の表示

### 4. 誤字脱字チェック
- よくある誤変換・誤入力をローカル辞書でチェック
- 完全オフライン対応（外部API不使用）

### 5. 日付整合性チェック
- 過去の日付への言及を検出
- 「明日」と過去日付が混在する矛盾を検出

### 6. 曜日整合性チェック
- 日付と曜日の不一致を検出
- 例：「12月30日（月）」と書いて実際は火曜日の場合に警告

## インストール方法

### 開発者モードでインストール

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトのフォルダを選択

## 使い方

1. Gmailを開く
2. 新規メールを作成
3. 送信ボタンをクリック
4. 確認ダイアログが表示される
5. 内容を確認して「送信」または「キャンセル」

## 設定

拡張機能アイコンをクリック → 「詳細設定」から以下の設定が可能：

- 拡張機能の有効/無効
- 各チェック機能の個別ON/OFF
  - 送信元チェック
  - 送信先チェック
  - 添付ファイルチェック
  - 誤字脱字チェック
  - 日付整合性チェック
  - 曜日整合性チェック

## プライバシー

- **すべての処理はローカルで実行**されます
- メールの内容は外部サーバーに送信されません
- 外部APIは使用していません

## ファイル構造

```
mail-bannin/
├── manifest.json              # Chrome拡張機能設定
├── _locales/ja/messages.json  # 日本語ローカライズ
├── src/
│   ├── background/
│   │   └── service-worker.js  # Service Worker
│   ├── content/
│   │   ├── content-script.js  # メインスクリプト
│   │   ├── gmail-dom.js       # Gmail DOM操作
│   │   ├── email-parser.js    # メール解析
│   │   └── styles.css         # ダイアログスタイル
│   ├── checkers/
│   │   ├── sender-checker.js     # 送信元チェック
│   │   ├── recipient-checker.js  # 送信先チェック
│   │   ├── attachment-checker.js # 添付チェック
│   │   ├── typo-checker.js       # 誤字脱字チェック
│   │   ├── date-checker.js       # 日付チェック
│   │   └── weekday-checker.js    # 曜日チェック
│   ├── ui/
│   │   └── confirmation-dialog.js # 確認ダイアログ
│   ├── options/               # 設定画面
│   └── popup/                 # ポップアップ
└── assets/
    ├── icons/                 # アイコン
    └── dictionaries/          # 辞書ファイル
```

## 技術仕様

- **Manifest Version**: 3
- **対応ブラウザ**: Chrome
- **対応サイト**: Gmail (mail.google.com)

## 既知の制限

- GmailのDOM構造変更により、一部機能が動作しなくなる可能性があります
- 複数のセレクタパターンでフォールバック対応済み
